# Online Polling Application

## Overview

The Online Polling Application is a full-stack web application that allows users to create polls and vote for different options. The system collects responses from users and displays poll results instantly. This project is developed using the MERN stack.

## Technologies Used

* MongoDB – Database for storing poll data
* Express.js – Backend framework for building APIs
* React.js – Frontend library for building user interfaces
* Node.js – Runtime environment for server-side development

## Features

* Create polls with multiple voting options
* Users can vote for their preferred option
* Real-time display of poll results
* Responsive and user-friendly interface
* Full-stack MERN architecture

## Installation and Setup

1. Clone the repository
   git clone https://github.com/shivalireddy-dev/online-polling-application.git

2. Navigate to the project folder
   cd online-polling-application

3. Install dependencies

Backend
cd backend
npm install

Frontend
cd frontend
npm install

4. Run the application

Backend
npm start

Frontend
npm start
